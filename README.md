# AQChatMobile

#### 介绍
AQChat 一个极速、便捷的匿名在线即时聊天室。
<br/>
<br/>
对标游戏后端开发，采用Netty作为通讯框架，支持高并发，高性能的即时通讯。全程无需HTTP协议，支持文本、图片、文件、音频、视频的发送和接收。
<br/>
<br/>
本项目为移动端代码，WEB端代码请查看：<a href="https://gitee.com/howcode/aq-chat">AQChat</a>，服务端代码请查看<a href="https://gitee.com/howcode/aq-chat-server">AQChat-Server</a>


项目地址：https://aqchat.run 【pc端访问默认访问WEB端地址,移动端访问跳转移动端地址】
